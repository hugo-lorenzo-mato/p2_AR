# FW
ip access-list extended FW
permit tcp any host 10.255.1.12 eq 80
permit tcp any host 10.255.2.12 eq 80
permit tcp any host 10.255.1.10 eq 443

int fa0/1
ip access-group FW in

ip inspect name INTERNET icmp 
ip inspect name INTERNET tcp 
ip inspect name INTERNET telnet 
ip inspect name INTERNET udp 
int fa0/0
ip inspect INTERNET in
